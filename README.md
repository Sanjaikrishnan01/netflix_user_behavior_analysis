# Netflix User Behaviour Dashboard 

A Power BI Dashboard analyzing Netflix viewing trends and patterns.  
---

## Description  
This project explores Netflix user behaviour using data analytics.  
The dashboard provides insights into viewing preferences, most-watched genres, ratings, and user activity patterns.  
---

## Features  
- Interactive Power BI dashboard  
- Genre-wise analysis of shows and movies  
- Trends in user ratings and preferences  
- Insights on content popularity across time  
---

## Tech Stack  
- **Tool:** Power BI  
- **Data:** Netflix dataset (CSV/Excel)  
- **Skills Used:** Data Cleaning, Data Visualization, Dashboard Building
